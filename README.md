# FaidatWorks — Personal Portfolio

A personal portfolio website showcasing my skills, projects, and background as a frontend developer and IT professional based in Nigeria.

## Sections
- **Hero** — introduction, skill tags, CTA buttons
- **About** — my story, education, quick facts
- **Projects** — filterable project cards with GitHub links
- **Skills** — Frontend, IT & Security, C Programming
- **Contact** — form, email, LinkedIn, GitHub, WhatsApp

## Tech Stack
React · Vite · Tailwind CSS · React Icons
